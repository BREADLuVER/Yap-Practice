# Yap Practice

Yap Practice is a small app for people who want to practice yapping using Northernlion clips.

## Intention

- Use short Northernlion clips as speaking prompts.
- Help users practice off-the-cuff speaking in a low-pressure way.
- Make practice sessions quick, simple, and repeatable.

## Project Structure

- `frontend`: Next.js app for the user-facing experience.
- `backend`: API and server-side logic.
